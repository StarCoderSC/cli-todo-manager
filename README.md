# 📝 CLI To-Do Manager

A Lightweight command-line to-do list app written in Pyton.
Build with love using just basic tools - no frameworks, no lag, no fluff.

## 🚀 Features
- Add tasks
- View all tasks
- Mark tasks as done
- Delete tasks
- Persistent storage via file system (tasks.txt)

## 🧰 How to Run
```bash
python todo_cli.py
